encode(prefix, type, hash)

Encodes a hash from a given type into a Bitcoin Cash address with the given prefix.
Parameters:
Name Type Description
prefix string Network prefix. E.g.: 'bitcoincash'.
type string Type of address to generate. Either 'P2PKH' or 'P2SH'.
hash Array Hash to encode represented as an array of 8-bit integers.
